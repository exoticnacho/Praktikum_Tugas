# Lumina Creative - Digital Agency Website

## Concept
**Lumina Creative** is a high-end digital design agency. The design focuses on:
- **Glassmorphism**: Translucent cards and navbar for a modern feel.
- **Gradients**: Vibrant purple and cyan accents against a dark background.
- **Typography**: Using 'Outfit' from Google Fonts for a clean, geometric look.
- **Animations**: Subtle floating and fade-in effects.

## How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser
